# 积分银行 · Kids Points Bank

> 把无形的"自律"变成有形的"资产"，用代币机制催化孩子的自发驱动力。

参考自 [杰西卡《暑假成长积分银行》](https://mp.weixin.qq.com/s/rgny628l633XrJeZokrcZg)，基于教育心理学中的**代币经济（Token Economy）**理论构建。

## 🧠 设计原理

| 问题 | 解法 | 心理学依据 |
|------|------|-----------|
| 孩子面对抽象、长期的成长目标缺乏动力 | 把每个行动变成即时可见的积分 | 即时正反馈循环，多巴胺驱动 |
| 大目标让人迷茫 | 拆解为每天看得见的小任务 | 目标梯度效应 |
| 打卡容易三分钟热度 | 连续打卡奖励 + 每周家庭复盘 | 反思性学习，防止倦怠 |
| 物质奖励削弱内在动机 | 经验型奖励为主（电影、旅行、选菜单） | 内在动机保护 |

## ✨ 五大模块

### 1. 🗺️ 成长地图
五大维度可视化，点击展开任务详情，完成度环形图实时展示。

### 2. ✅ 今日打卡
- 按维度分类的任务列表
- 勾选即赚积分，实时飘字动画 + 震动反馈
- 连续打卡里程碑奖励（3/7/14/30天）
- 每日打卡上限防沉迷

### 3. 🎁 奖励兑换
三级阶梯奖励体系：
| 等级 | 价格 | 示例 |
|------|------|------|
| ⭐ 小奖励 | 10分 | 选一次家庭电影、点播睡前故事 |
| 🌟 中奖励 | 30分 | 买一本喜欢的书、一次亲子外出 |
| 💫 大奖励 | 60分 | 一次短途旅行、实现一个小愿望 |

家长可自定义奖励内容，兑换需家长确认。

### 4. 📊 每周复盘
周末家庭会议模板：
- 孩子写"最棒的事"和"遇到的困难"
- 家长写"看见的进步"
- 孩子提"希望的支持"
- 自动统计本周任务完成数、积分、点亮维度

### 5. 📸 成长作品
- 拍照/上传照片记录成长瞬间
- 关联成长维度标签
- 图片自动压缩存储（IndexedDB）
- 期末自动生成成长档案

## 🛡️ 安全与隐私

- **纯前端应用**：所有数据存储在本地浏览器（localStorage + IndexedDB）
- **零网络请求**：无需服务器，无需注册，离线可用
- **家长门禁**：奖励兑换、积分修正需家长确认
- **数据导出**：一键导出 JSON 备份，随时恢复
- **XSS 防护**：所有用户输入经过 HTML 转义

## 📱 使用方法

### 快速开始
1. 在浏览器中打开 `index.html`
2. 首次打开会显示欢迎页
3. 点击底部导航切换模块

### 家长设置
- 奖励兑换需要家长确认（弹窗二次确认）
- 可在 `js/data.js` 中自定义任务、积分、奖励内容
- 可在 `js/storage.js` 的 `DEFAULT_DATA.settings` 中设置 PIN 码

### 数据备份
进入「成长作品」页面 → 使用浏览器开发者工具调用：
```javascript
PointsBankStorage.exportAllData()
```
导出为 JSON 文件，可随时用 `importAllData(jsonString)` 恢复。

## 🔧 技术栈

- **零依赖**：纯 HTML + CSS + Vanilla JS
- **存储**：localStorage（结构化数据）+ IndexedDB（图片/媒体）
- **响应式**：移动端优先，支持桌面端自适应
- **无障碍**：支持 `prefers-reduced-motion`，打印友好

## 🐛 已知限制

- 数据存储在浏览器本地，换设备需手动导出/导入
- 图片压缩质量固定为 70% JPEG
- 不支持多孩子/多账户
- 家长设置面板尚未实现（需手动编辑代码）

## 📄 许可

本项目仅供家庭使用，非商业用途。
