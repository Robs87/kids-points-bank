## [1.0.0] - 2026-06-30

### Added
- 五大成长维度：学习力、运动力、自控力、探索力、实践力
- 成长地图：可视化任务进度，环形进度图展示
- 今日打卡：勾选即赚积分，实时飘字动画 + 震动反馈
- 连续打卡奖励：3/7/14/30 天里程碑奖励
- 奖励兑换：三级阶梯（10/30/60 分），经验型奖励为主
- 每周复盘：家庭会议模板，双向沟通
- 成长作品：照片/视频上传，IndexedDB 存储
- 家长门禁：PIN 码 + 二次确认
- Docker 部署支持（GHCR 镜像）
- Unraid 模板部署支持
- 数据导出/导入备份
- XSS 防护（用户输入转义）
- 图片自动压缩（IndexedDB 存储）
- 响应式设计，移动端优先
- 无障碍支持（prefers-reduced-motion）
- 中英文双语 README
