# Security Policy

## Supported Versions

本项目为纯前端静态应用，不涉及后端服务、数据库或 API 密钥。所有数据存储在用户本地浏览器中。

| 版本 | 支持状态 |
|------|---------|
| 1.0.0+ | ✅ 当前版本 |

## Reporting a Vulnerability

由于本项目是纯前端应用（HTML/CSS/JS），不存在服务端漏洞。如发现安全问题，请通过以下方式报告：

1. **GitHub Issues** — 设置为私密 issue（Private）
2. **GitHub Security Advisories** — 通过仓库页面的 "Security" → "Advisories" 报告

我们将在 48 小时内回复。

## 安全注意事项

- 本项目不涉及任何网络请求，所有数据存储在本地
- 如需部署到公共服务器，建议使用 HTTPS
- 家长应妥善保管浏览器数据，积分和成长记录仅存储在本地
